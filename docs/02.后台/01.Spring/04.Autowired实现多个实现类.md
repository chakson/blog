# Autowired实现多个实现类

@Autowired 如何实现多个实现类

<!-- more -->

## 解决方案

```java
@Service
@Qualifier("a")
public class AService implements Service {}

@Service
@Qualifier("b")
public class BService implements Service {}
```

```java
@Qualifier("a")
@Autowired
Service aService;

@Qualifier("b")
@Autowired
Service bService;
```
